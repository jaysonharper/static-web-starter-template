Flow Components Documentation

Flow Button

Reusable button component based on Flowbite design patterns.

Variants

Primary Secondary Success Danger Warning

Sizes

XS SM MD LG XL

States

Disabled Loading

Usage

<flow-button variant="primary" size="md">Click me</flow-button> <flow-button variant="success" disabled>Disabled</flow-button> <flow-button loading>Processing...</flow-button>

Properties

Property Type Default Description
variant String 'primary' Button color variant
size String 'md' Button size
disabled Boolean false Disables the button
loading Boolean false Shows loading spinner

Flow Alert

Alert component for displaying important messages.

Types

Info: This is an info alert with dismissible option. Success: Operation completed successfully! Warning: Please check your input. Error: Something went wrong.

Usage

<flow-alert type="success" dismissible> <strong>Success!</strong> Your action was completed. </flow-alert>
